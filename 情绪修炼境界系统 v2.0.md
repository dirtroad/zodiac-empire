# 星座帝国 - 情绪修炼境界系统策划

> **版本：** v2.0（深度沉迷版）  
> **日期：** 2026-03-17  
> **策划：** 小策

---

## 一、核心设计理念

### 让人沉迷的 5 大机制

**1. 即时反馈** - 每次修炼都有 visible 提升
**2. 随机惊喜** - 修炼结果有概率触发特殊效果
**3. 损失厌恶** - 中断修炼会损失进度
**4. 社交攀比** - 境界全服可见，激发竞争
**5. 沉没成本** - 投入越多越舍不得放弃

---

## 二、修炼境界体系

### 2.1 境界等级（15 阶）

```
凡人境（1-3 重）→ 入门
  ↓
修士境（4-6 重）→ 小成
  ↓
真人境（7-9 重）→ 大成
  ↓
仙人境（10-12 重）→ 化境
  ↓
圣人境（13-15 重）→ 巅峰
```

### 2.2 每重境界效果

| 境界 | 战力加成 | 特殊效果 | 修炼难度 |
|-----|---------|---------|---------|
| 凡人 1-3 重 | +1%/+2%/+3% | 无 | 低（情绪×10/重） |
| 修士 4-6 重 | +5%/+7%/+10% | 每日自动产出情绪×10 | 中（情绪×50/重） |
| 真人 7-9 重 | +12%/+15%/+20% | 战斗胜利额外 +5% 情绪 | 高（情绪×200/重） |
| 仙人 10-12 重 | +25%/+30%/+40% | 修炼速度 +50% | 极高（情绪×1000/重） |
| 圣人 13-15 重 | +50%/+70%/+100% | 解锁专属技能 | 传说（情绪×5000/重） |

---

## 三、修炼方式（让人上瘾的核心）

### 3.1 基础修炼（每日必做）

```
消耗情绪资源 → 获得修炼值 → 提升境界

消耗比例：
· 喜悦：修炼值×1.0（基础）
· 平和：修炼值×1.2（防御向）
· 悲伤：修炼值×1.5（战术向）
· 愤怒：修炼值×2.0（攻击向）
· 恐惧：修炼值×2.5（闪避向）
· 贪婪：修炼值×3.0（金币向，但境界提升慢）
```

**沉迷机制：**
- 每日首次修炼 ×2 倍效果（限时 24 小时刷新）
- 连续修炼 7 天额外奖励（损失厌恶）
- 修炼进度条可视化（只差一点就升级）

---

### 3.2 闭关修炼（大额投入）

```
选择闭关时长：
· 1 小时：情绪×100，修炼值×100
· 4 小时：情绪×350，修炼值×400（省 12.5%）
· 8 小时：情绪×600，修炼值×800（省 25%）
· 24 小时：情绪×1500，修炼值×2400（省 37.5%）
```

**沉迷机制：**
- 闭关期间不能做其他事（沉没成本）
- 可以提前出关，但损失 50% 进度（损失厌恶）
- 闭关结束有概率触发"顿悟"（随机惊喜）

---

### 3.3 顿悟系统（随机惊喜）

```
顿悟触发概率：1%-10%（根据境界）

顿悟效果：
· 小顿悟：修炼值×2（持续 1 次修炼）
· 大顿悟：修炼值×5（持续 1 次修炼）
· 天降祥瑞：直接提升 1 重境界（极稀有）
```

**沉迷机制：**
- 每次修炼都有概率触发（赌博心理）
- 全服公告天降祥瑞（社交攀比）
- 顿悟时特效拉满（视觉刺激）

---

### 3.4 心魔系统（失败惩罚）

```
心魔触发条件：
· 修炼失败（概率根据境界）
· 中断闭关
· 连续 3 天不修炼

心魔效果：
· 修炼值 -10%
· 战力 -5%
· 持续 24 小时
```

**沉迷机制：**
- 害怕失去所以每天上线（损失厌恶）
- 需要消耗情绪驱除心魔（继续投入）
- 心魔期间修炼效果减半（焦虑感）

---

### 3.5 渡劫系统（关键节点）

```
需要渡劫的境界：
· 凡人→修士（第 3 重→第 4 重）
· 修士→真人（第 6 重→第 7 重）
· 真人→仙人（第 9 重→第 10 重）
· 仙人→圣人（第 12 重→第 13 重）

渡劫成功率：
· 基础成功率：50%
· 消耗情绪可提高成功率（每×100 情绪 +10%）
· 渡劫失败：境界下降 1 重
```

**沉迷机制：**
- 渡劫前紧张刺激（情绪投入）
- 失败损失巨大（损失厌恶）
- 成功全服公告（社交炫耀）
- 可以购买"渡劫保护符"（付费点）

## 四、社交攀比系统

### 4.1 境界排行榜

```
· 全服境界榜（前 100 名）
· 好友境界对比（实时可见）
· 本服圣人数量统计
```

**攀比机制：**
- 排行榜每日结算，前 10 名发放奖励
- 好友之间显示"你已超过 XX%"
- 境界变化推送通知

---

### 4.2 境界威压

```
高境界对低境界的压制：

· 圣人境：对凡人境战力 +50%
· 仙人境：对修士境战力 +30%
· 真人境：对凡人境战力 +20%
```

**攀比机制：**
- 被高境界玩家攻击时显示"威压"提示
- 激发"我要变强"的动力
- 低境界玩家可以通过组队抵消威压

---

### 4.3 境界特效

```
· 凡人境：无特效
· 修士境：周身微光
· 真人境：脚下光环
· 仙人境：身后光翼
· 圣人境：全身金光 + 特殊 BGM
```

**攀比机制：**
- 特效全服可见
- 进入任何场景都有特效展示
- 可以关闭特效，但 99% 的人不会关

---

## 五、免费保护机制（无付费设计）

### 5.1 渡劫保护

```
· 每日首次渡劫必成功（新手保护）
· 连续修炼 7 天赠送"渡劫福缘"（保底成功）
· 好友助力可提高成功率（社交玩法）
· 渡劫失败保护：境界不下降，但需要冷却 24 小时
```

### 5.2 心魔驱除

```
· 登录即送"清心丹"×1（每日）
· 好友帮忙驱除心魔（社交互动）
· 连续修炼 3 天自动驱除心魔
```

### 5.3 修炼加速

```
· 每日首次修炼×2 倍效果（免费）
· 观看广告可获得修炼加速（可选）
· 好友助力修炼速度 +50%（社交玩法）
```

## 六、日常任务绑定

### 6.1 修炼日常

```
每日任务：
· 修炼 1 次：金币×100
· 修炼 3 次：金币×300 + 情绪×50
· 修炼 5 次：金币×500 + 情绪×100
· 连续修炼 7 天：稀有情绪×1
```

**沉迷机制：**
- 任务进度可视化（还差 2 次就完成了）
- 连续奖励越来越丰厚
- 断签会损失累计进度

---

### 6.2 境界任务

```
境界提升任务：
· 突破到修士境：钻石×50
· 突破到真人境：钻石×100 + 称号"真人"
· 突破到仙人境：钻石×200 + 称号"仙人"
· 突破到圣人境：钻石×500 + 称号"圣人" + 全服公告
```

## 七、技术实现

### 7.1 数据结构

```typescript
interface Cultivation {
  userId: number;
  realm: number;        // 1-15（境界）
  level: number;        // 1-3（重数）
  cultivationValue: number;  // 修炼值
  lastCultivateTime: Date;   // 最后修炼时间
  consecutiveDays: number;   // 连续修炼天数
  hasHeartDemon: boolean;    // 是否有心魔
  breakthroughProgress: number;  // 渡劫进度（0-100）
}
```

### 7.2 核心接口

```typescript
// 修炼
POST /v1/cultivation/cultivate
{
  emotionType: number,
  amount: number
}

// 闭关
POST /v1/cultivation/seclude
{
  duration: number  // 1/4/8/24 小时
}

// 渡劫
POST /v1/cultivation/tribulation

// 驱除心魔
POST /v1/cultivation/remove-heart-demon

// 查询境界
GET /v1/cultivation/status

// 境界排行榜
GET /v1/cultivation/ranking
```

## 十、风险控制

### 10.1 数值平衡

**风险：** 境界提升太快，后期无内容
**控制：**
- 每日修炼次数上限（免费玩家 5 次/天）
- 境界越高提升越慢（指数增长）
- 设置境界天花板（当前版本圣人 15 重）

### 10.2 付费平衡

**风险：** 付费玩家碾压免费玩家
**控制：**
- 免费玩家通过时间可以追上
- 组队可以抵消境界差距
- 活动赠送渡劫保护符
