## 一、战力系统

### 1.1 基础战力公式

```
总战力 = 基础战力 + 装备战力 + 情绪加成 + 星座加成 + 联盟加成
```

### 1.2 详细计算

#### 基础战力
```
基础战力 = 等级 × 100 + 星系战力 + 建筑战力

等级战力 = 等级 × 100
星系战力 = Σ(星系等级 × 50 × 星系稀有度系数)
建筑战力 = Σ(建筑等级 × 20)

稀有度系数：
- 普通：1.0
- 稀有：1.5
- 史诗：2.0
- 传说：3.0
```

#### 装备战力
```
装备战力 = Σ(装备基础战力 × (1 + 强化加成) × 稀有度系数)

装备基础战力：
- 普通装备：10-30
- 稀有装备：30-80
- 史诗装备：80-200
- 传说装备：200-500
- 神话装备：500-1500

强化加成：
- +1: +5%
- +2: +12%
- +3: +20%
- +4: +30%
- +5: +42%
- +6: +56%
- +7: +72%
- +8: +90%
- +9: +110%
- +10: +132%

稀有度系数：
- 普通：1.0
- 稀有：1.3
- 史诗：1.6
- 传说：2.0
- 神话：3.0
```

#### 情绪加成
```
情绪战力加成 = 快乐×0.5 + 激情×1.0 + 智慧×0.3 + 愤怒×1.5

说明：
- 快乐：每点+0.5战力（辅助型）
- 激情：每点+1.0战力（攻击型）
- 智慧：每点+0.3战力（策略型，但有其他加成）
- 愤怒：每点+1.5战力（高风险高收益）

注意：愤怒情绪超过500时，防御力-20%
```

#### 星座加成
```
星座战力加成 = 基础战力 × 星座加成系数

星座加成系数：
- 白羊座：+5% 战力
- 金牛座：+10% 产出
- 双子座：+8% 贸易收益
- 巨蟹座：+5% 防御
- 狮子座：+10% 联盟加成
- 处女座：+5% 合成成功率
- 天秤座：+3% 全属性
- 天蝎座：+8% 暴击率
- 射手座：+20% 行动力恢复
- 摩羯座：+15% 时间利息
- 水瓶座：+5% 稀有装备概率
- 双鱼座：+10% 情绪采集
```

#### 星座匹配加成（新增）
```
好友组队时，根据星座匹配度获得额外战力加成：

匹配度判定（四元素）：
- 最佳匹配：火↔风、土↔水 → 战力+20%
- 良好匹配：同元素（火↔火等）→ 战力+10%
- 普通匹配：其他组合 → 战力+5%

示例：
白羊座（火）与双子座（风）组队
→ 最佳匹配 → 双方战力各+20%

白羊座（火）与狮子座（火）组队
→ 良好匹配 → 双方战力各+10%
```

#### 联盟加成
```
联盟战力加成 = 基础战力 × 联盟等级 × 0.5%

联盟等级加成：
- Lv.1: +0.5%
- Lv.2: +1.0%
- Lv.3: +1.5%
- Lv.4: +2.0%
- Lv.5: +2.5%
...
- Lv.10: +5.0%
```

### 1.3 战力示例

**玩家A（Lv.10，白羊座）：**
```
基础战力 = 10 × 100 + 2×50×1.5 + 3×20 = 1210
装备战力 = (50×1.2×1.3) + (30×1.0×1.0) = 78 + 30 = 108
情绪加成 = 100×0.5 + 50×1.0 = 100
星座加成 = (1210+108+100) × 5% = 71

总战力 = 1210 + 108 + 100 + 71 = 1489
```

**玩家A与玩家B（双子座）组队：**
```
白羊座（火）与双子座（风）= 最佳匹配

玩家A获得战力加成：1489 × 20% = 298
组队战力 = 1489 + 298 = 1787
```

### 1.4 战队战力计算（新增）

```
战队总战力 = Σ(成员战力) × 匹配加成系数

匹配加成系数计算：
1. 统计所有成员两两星座匹配度
2. 计算各等级匹配占比
3. 加权求和

公式：
匹配加成系数 = 最佳匹配占比×0.20 + 良好匹配占比×0.10 + 普通匹配占比×0.05
```

**战队示例（5人）：**
```
成员：
- 玩家A：白羊座（火）
- 玩家B：双子座（风）→ 与A最佳匹配
- 玩家C：狮子座（火）→ 与A良好匹配
- 玩家D：天秤座（风）→ 与B良好匹配，与A最佳匹配
- 玩家E：射手座（火）→ 与A、C良好匹配，与B、D最佳匹配

两两关系共10对：
最佳匹配：A-B, A-D, B-E, D-E = 4对（40%）
良好匹配：A-C, B-D, C-E = 3对（30%）
普通匹配：B-C, C-D = 2对（20%）

匹配加成系数 = 40%×0.20 + 30%×0.10 + 20%×0.05
             = 0.08 + 0.03 + 0.01
             = 0.12（+12%）

假设成员战力：1500 + 1800 + 1600 + 1400 + 1700 = 8000
战队总战力 = 8000 × 1.12 = 8960
```

**战队战力对比：**
```
未优化的战队（无匹配加成）：8000 战力
优化后的战队（+12%加成）：8960 战力

差距：960 战力（相当于多一个成员）
```

## 二、战斗系统

### 2.1 战斗判定

```
胜率 = 我方战力 / (我方战力 + 敌方战力) × 100%

示例：
我方战力：1500
敌方战力：1000
胜率 = 1500 / (1500 + 1000) × 100% = 60%
```

### 2.2 战斗回合

```
回合数 = 10 + 随机(0, 10) = 10-20回合

每回合伤害 = 攻击方战力 × 伤害系数 × 暴击系数

伤害系数 = 0.05 + 随机(-0.02, 0.02) = 3%-7%

暴击系数：
- 普通攻击：1.0
- 暴击：1.5（暴击率由装备和情绪决定）
- 闪避：0（闪避率由装备决定）
```

### 2.3 占星学星座关系系统（新增）

#### 元素相生（团队加成）

| 关系 | 团队战力加成 | 适用场景 |
|-----|-------------|---------|
| **同元素** | +20% | 战队、联盟、好友组队 |
| **风助火** | +15% | 风象+火象玩家合作 |
| **水润土** | +15% | 水象+土象玩家合作 |
| **其他组合** | +5% | 其他元素组合 |

**注意：相生关系仅在团队合作时生效，战斗攻击中无加成**

#### 对宫星座（战斗对抗）

| 对宫组合 | 元素组合 | 战斗伤害加成 |
|---------|---------|-------------|
| 白羊 ↔ 天秤 | 火 ↔ 风 | +30% |
| 金牛 ↔ 天蝎 | 土 ↔ 水 | +30% |
| 双子 ↔ 射手 | 风 ↔ 火 | +30% |
| 巨蟹 ↔ 摩羯 | 水 ↔ 土 | +30% |
| 狮子 ↔ 水瓶 | 火 ↔ 风 | +30% |
| 处女 ↔ 双鱼 | 土 ↔ 水 | +30% |

#### 极性竞争（战斗加成）

| 极性组合 | 战斗伤害加成 | 说明 |
|---------|-------------|------|
| **同极性对抗** | +10% | 主动vs主动、固定vs固定、变动vs变动 |

#### 元素相克（战斗加成）

| 元素关系 | 战斗伤害加成 | 说明 |
|---------|-------------|------|
| 火 ↔ 水 | +5% | 相冲元素 |
| 土 ↔ 风 | +5% | 相冲元素 |

#### 星座关系计算示例

**团队战力计算：**
```
战队5人：
- 白羊座（火）
- 狮子座（火）→ 与白羊同元素 +20%
- 双子座（风）→ 与白羊风助火 +15%，与狮子风助火 +15%
- 天秤座（风）→ 与双子同元素 +20%，与白羊风助火 +15%
- 射手座（火）→ 与白羊、狮子同元素 +20%

平均加成 = (20+20+15+15+20+15+15+20+20+20) / 10 = 18%
战队总战力 = Σ成员战力 × 1.18
```

**战斗伤害计算：**
```
白羊座（火）vs 天秤座（风）
关系：对宫星座
伤害加成：+30%

白羊座攻击：基础伤害 1000 × 1.30 = 1300
天秤座反击：基础伤害 900 × 1.30 = 1170

双方都受到额外伤害！
```

### 2.4 星座匹配加成

```
最佳匹配：战力+20%，贸易收益+20%
良好匹配：战力+10%，贸易收益+10%
普通匹配：战力+5%，贸易收益+5%

匹配规则（四元素）：
- 火象（白羊、狮子、射手）：最佳匹配风象，良好匹配火象
- 土象（金牛、处女、摩羯）：最佳匹配水象，良好匹配土象
- 风象（双子、天秤、水瓶）：最佳匹配火象，良好匹配风象
- 水象（巨蟹、天蝎、双鱼）：最佳匹配土象，良好匹配水象
```

### 2.4 战斗奖励

```
胜利奖励选项：
1. 星系：随机1个敌方卫星星系（概率30%）
2. 装备：随机1件敌方装备（概率25%）
3. 资源：金币×(敌方等级×50) + 随机情绪资源×50

复仇加成：
- 复仇胜利：奖励×2
- 复仇失败：无惩罚
```

```
利息 = 本金 × 日利率 × 天数

利息 = 1000 × 0.05 × 7 = 350晶体
```

```
- 信用分 601-700：+1%
- 信用分 701-800：+2%
- 信用分 801-900：+3%
- 信用分 901-1000：+5%

计算公式：
利息 = 本金 × 日利率 × 天数

示例：
存款：1000时间币
利率：5%/日
存期：7天
利息 = 1000 × 0.05 × 7 = 350时间币
```

### 6.3 贷款利率

| 贷款期限 | 日利率 | 最高额度 |
|---------|-------|---------|
| 1-3天 | 8% | 500时间币 |
| 4-7天 | 10% | 1000时间币 |
| 8-14天 | 12% | 2000时间币 |
| 15-30天 | 15% | 5000时间币 |

```
还款金额 = 本金 × (1 + 日利率 × 天数)

示例：
贷款：1000时间币
利率：10%/日
期限：7天
还款 = 1000 × (1 + 0.10 × 7) = 1700时间币
```

```
- 担保成功：+5分/次

信用分影响：
- 500-600：正常利率
- 601-700：利率-10%
- 701-800：利率-20%
- 801-900：利率-30%
- 901-1000：利率-50%，额度+50%
```

## 九、平衡性设计

### 9.1 战力平衡

```
等级压制限制：
- 高10级以内：正常战斗
- 高10-20级：战力差异缩小20%
- 高20级以上：战力差异缩小50%

防止小号虐大号机制。
```

### 9.2 资源平衡

```
产出消耗比：
- 金币：产出:消耗 = 1.2:1（略盈余）
- 钻石：产出:消耗 = 0.3:1（需充值）
- 时间币：产出:消耗 = 1:1（平衡）

每日产出上限：
- 金币：5000
- 时间币：500
- 情绪资源：每种100
```

### 9.3 时间平衡

```
关键时间点：
- 建筑建设：1-8小时
- 建筑升级：2-24小时
- 资源收取：每4小时
- 护盾时长：12小时
- 行动力恢复：1点/小时

在线时长期望：1-2小时/天
```
