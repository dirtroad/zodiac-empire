# 星座帝国 - UI原型流程图

> 关键页面流程与交互设计 v1.0

---

## 一、整体信息架构

```
启动页
    ↓
登录/注册（微信一键登录）
    ↓
星座选择（首次登录）
    ↓
┌─────────────────────────────────────────────────────┐
│                    主界面（TabBar）                   │
├─────────────────────────────────────────────────────┤
│  首页  │  星系  │  战斗  │  市场  │  我的  │
└─────────────────────────────────────────────────────┘
    ↓        ↓        ↓        ↓        ↓
   详情     详情     详情     详情     详情
```
