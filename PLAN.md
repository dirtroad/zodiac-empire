# 星座帝国 - 微信小游戏产品计划书

> 你的情绪，你的时间，你的朋友圈，都是你的战场

---

## 一、产品概述

### 1.1 核心概念
每个微信用户拥有一个独特的"星座帝国"，好友关系形成"星际航道"。玩家经营宇宙、采集情绪资源、管理时间货币、与其他玩家贸易或战争，抢夺星系领地和稀有装备。

### 1.2 六大系统融合

| 原方案 | 融合后定位 |
|-------|-----------|
| 情绪交易所 | 情绪成为核心资源，可采集、加工、交易、战争掠夺 |
| 情绪卡片 | 玩家上传聊天截图/视频，路人点赞投票，人气值转化为情绪资源 |
| 时间银行 | 时间是货币，可存取、借贷、投资、加速建设 |
| 星座星域 | 根据星座划分出生区域，影响属性加成和社交匹配 |
| 关系图谱经营 | 好友关系 = 星际航道，好友互动 = 航道流量 = 收益 |
| 平行世界贸易站 | 每个玩家宇宙有独特资源，差异化产生贸易需求 |
| 领地战争 | 星系可抢夺，装备可合成升级，所有物品可交易 |

### 1.3 一句话定位
"把你的情绪、时间、朋友圈变成一个宇宙帝国"

## 二、世界观设定

### 2.1 星座帝国
- 每个玩家拥有一个独立的"星座帝国"
- 宇宙由多个"情绪星系"组成
- 好友之间的连线形成"星际航道"
- 情绪资源分布在各个星系

### 2.2 情绪资源体系

| 情绪类型 | 颜色 | 用途 | 获取方式 |
|---------|------|------|---------|
| 快乐 | 金色 | 加速建设、提升战力 | 好友互动、战斗胜利 |
| 平静 | 蓝色 | 稳定产出、防御加成 | 每日签到、冥想 |
| 激情 | 红色 | 攻击加成、合成强化 | 战斗、竞技场 |
| 智慧 | 紫色 | 科技研发、策略加成 | 时间投资、研究 |
| 愤怒 | 黑色 | 特殊武器、高攻击但高风险 | 被抢、复仇 |
| **恐惧** | 深紫色 | 防御穿透、闪避加成 | **地盘被抢夺时产生** |
| 焦虑 | 灰色 | 可加工成专注药水 | 自动产生，需转化 |

#### 恐惧情绪特殊机制

```
恐惧情绪：
- 触发条件：自己的地盘/建筑物被其他玩家抢夺
- 获取量：根据被抢夺地盘的价值决定（价值越高，恐惧越多）
- 特殊效果：
  · 防御穿透+10%（攻击时无视对方10%防御）
  · 闪避加成+5%（战斗中有5%概率闪避攻击）
  · 可用于合成"暗影系"装备

恐惧转化：
- 恐惧可以转化为愤怒（复仇时使用）
- 转化比例：1恐惧 = 2愤怒
- 在复仇战役中，恐惧+愤怒组合可触发"恐惧复仇"效果（伤害+50%）
```

### 2.3 情绪卡片系统

#### 核心概念
玩家自愿上传聊天截图/视频，生成"情绪卡片"，路人点赞投票形成人气值，转化为游戏内情绪资源。

#### 情绪卡片生成
```
玩家上传聊天截图/视频 → AI识别情绪类型 → 生成情绪卡片 
    ↓
路人点赞投票 → 形成人气值 → 转化为稀有情绪资源
```

#### 功能模块
| 功能 | 说明 |
|-----|------|
| 上传入口 | 支持图片、短视频，玩家自愿上传 |
| 情绪识别 | AI自动识别情绪类型（快乐、激情、平静等） |
| 隐私保护 | 自动打码敏感信息，支持匿名发布 |
| 人气投票 | 全服玩家浏览卡片，点赞投票 |
| 人气转化 | 高人气卡片转化为稀有情绪资源 |

#### 人气值规则
| 人气等级 | 点赞数要求 | 转化奖励 |
|---------|-----------|---------|
| 普通 | 10+ | 10点基础情绪资源 |
| 热门 | 100+ | 50点 + 稀有情绪碎片 |
| 爆款 | 500+ | 200点 + 史诗情绪结晶 |
| 现象级 | 2000+ | 1000点 + 传说情绪核心 |

#### 社交玩法
- **情绪广场**：全服热门情绪卡片展示
- **情绪PK**：两张卡片PK，票数多的获胜
- **情绪收藏**：收藏喜欢的卡片，形成个人博物馆
- **情绪盲盒**：随机抽取匿名情绪卡片
- **分享拉票**：分享卡片到朋友圈，邀请好友点赞

#### 内容审核
- AI+人工双重审核
- 过滤违规内容（涉黄、涉政、广告等）
- 用户举报机制

### 2.5 时间货币体系

- **时间币**：核心货币，通过游戏时长积累
- **时间存取**：存入时间获得利息，借出时间支付利息
- **时间投资**：投资好友项目，分享收益
- **时间加速**：消耗时间币加速建设/生产

## 三、核心玩法

### 3.1 星系领地系统

#### 星系分配
- 每个微信用户自动获得一个"主星系"
- 可扩张占领更多"卫星星系"（最多9个）
- 星系类型决定情绪资源产出

| 星系类型 | 主产资源 | 稀有度 |
|---------|---------|--------|
| 快乐星系 | 快乐、金币 | 普通 |
| 智慧星系 | 智慧、科技点 | 稀有 |
| 激情星系 | 激情、战力 | 稀有 |
| 平静星系 | 平静、防御 | 史诗 |
| 混沌星系 | 愤怒、特殊资源 | 传说 |

#### 星系建设
- 在星系上建造"情绪工厂"加工资源
- 建造"时间银行"管理时间货币
- 建造"防御阵列"抵御攻击
- 建造"贸易站"与好友交易

### 3.2 星际航道系统

#### 航道生成
- 每个微信好友自动生成一条"星际航道"
- 好友互动（访问、点赞、送礼）增加航道流量
- 航道流量决定贸易收益

#### 航道等级
| 航道等级 | 流量要求 | 效果 |
|---------|---------|------|
| 量子通道 | 初始 | 基础贸易 |
| 曲速航道 | 100流量 | 贸易收益+20% |
| 跃迁通道 | 500流量 | 贸易收益+50%，可驻军 |
| 传送门 | 2000流量 | 瞬间到达，联合防御 |

### 3.3 装备系统

#### 装备类型
- **情绪引擎**：提升情绪采集效率
- **时间加速器**：减少建设/生产时间
- **星际武器**：攻击力加成
- **护盾发生器**：防御力加成
- **特殊装备**：独特效果（隐身、时间暂停等）

#### 装备品质
| 品质 | 颜色 | 战力加成 | 获取方式 |
|-----|------|---------|---------|
| 普通 | 白色 | +10 | 初始赠送、战斗掉落 |
| 稀有 | 蓝色 | +30 | 合成、市场购买 |
| 史诗 | 紫色 | +100 | 高级合成、活动 |
| 传说 | 橙色 | +300 | 顶级合成、赛季奖励 |
| 宇宙 | 红色 | +1000 | 极低概率、限时活动 |

#### 装备合成
```
3件普通 → 1件稀有（80%成功）
3件稀有 → 1件史诗（50%成功）
3件史诗 → 1件传说（20%成功）
3件传说 → 1件宇宙（5%成功）
```
- 合成可加入"情绪催化剂"提升成功率
- 失败后装备降级或消失（可用保护符）

### 3.4 战斗系统

#### 战斗类型
| 类型 | 说明 | 奖励 |
|-----|------|------|
| 星际战争 | 攻打好友抢星系 | 星系/装备/资源 |
| 情绪对决 | 纯装备战力比拼 | 装备/情绪资源 |
| 贸易争夺 | 抢夺航道控制权 | 航道收益分成 |
| 复仇战役 | 被抢后发起复仇 | 双倍抢回+额外奖励 |

#### 战斗机制
- 双方战力对比（基础战力 + 装备加成 + 情绪加成）
- 战斗自动进行，可观看回放
- 胜利选择奖励：1个星系 或 1件装备 或 资源

#### 防守机制
- 被攻击后获得12小时"情绪护盾"
- 好友可"驻军"帮助防守
- 建造防御阵列增加防守战力

### 3.5 贸易系统

#### 星际市场
- 所有玩家可在市场挂牌出售：星系、装备、情绪资源、时间币
- 自由定价，系统收取5%手续费
- 支持一口价、拍卖、物物交换

#### 好友贸易
- 通过星际航道与好友直接贸易
- 贸易收益与航道流量挂钩
- 可签订长期贸易协议

#### 情绪交易所
- 情绪资源实时价格波动
- 可低买高卖赚取差价
- 价格受全服情绪产出影响

### 3.10 时空晶体系统

#### 核心概念
时间不再只是货币，而是**穿梭宇宙的能量**！玩家通过游戏时长积累时空晶体，用于各种时空操作。

#### 时空晶体获取

| 方式 | 获得晶体 |
|-----|---------|
| 游戏时长 | 1分钟 = 1晶体 |
| 每日登录 | +50晶体 |
| 完成任务 | +10-100晶体 |
| 星系产出 | 建造时空能量站后产出 |
| 充值购买 | 1元 = 10晶体 |

#### 时空晶体用途

| 用途 | 消耗晶体 | 说明 |
|-----|---------|------|
| **时空穿梭** | 50-200 | 瞬间移动到其他星域 |
| **星座匹配搜索** | 10-50 | 快速找到匹配度高的玩家 |
| **加速建设** | 10-100 | 缩短建筑建设时间 |
| **加速生产** | 10-50 | 缩短资源产出时间 |
| **时空回溯** | 100 | 重看历史战斗，分析策略 |
| **时空预言** | 200 | 预览即将发生的战斗结果 |
| **时空冻结** | 500 | 冻结敌方星系1小时 |

#### 星座匹配搜索

```
使用时空晶体快速搜索匹配玩家：

搜索类型：
- 搜索最佳匹配 → 30晶体（找到星座最佳匹配的玩家）

- 组建战队
```

#### 时空穿梭功能

```
选择目标星域，消耗时空晶体瞬间移动：

🔥 火象星域  → 50晶体
🌍 土象星域  → 80晶体
💨 风象星域  → 100晶体
💧 水象星域  → 120晶体

特殊区域：
🌟 中央星域  → 200晶体
⚫ 黑洞边缘  → 500晶体
```

#### 时空回溯
- 重看历史战斗
- 分析战斗策略
- 复仇时获得战术优势

#### 时空预言
- 预览即将发生的战斗结果
- 帮助决定是否发起攻击
- 高消耗，战略技能

#### 时空冻结
- 冻结敌方星系
- 对方1小时内无法操作
- 高消耗，战略技能

| 操作 | 说明 |
|-----|------|
| 存入晶体 | 获得利息（日利率5%） |
| 借贷晶体 | 支付利息，预支能量 |

#### 时间存取
| 操作 | 说明 |
|-----|------|
| 存入时间 | 游戏时长转化为时间币，获得利息 |
| 借出时间 | 预支时间币，需支付利息 |
| 时间投资 | 投资好友项目，分享收益 |
| 时间加速 | 消耗时间币加速建设/生产 |

- 借贷需在规定时间偿还
- 逾期不还：星系被冻结，装备被扣押
- 可申请好友担保

### 3.7 占星学星座关系系统

#### 核心理论
基于占星学的四大元素（火、土、风、水）和三种极性（主动、固定、变动）设计星座相生相克关系。

#### 四大元素

| 元素 | 星座 | 特性 |
|-----|------|------|
| 🔥 火 | 白羊、狮子、射手 | 热情、行动、直觉 |
| 🌍 土 | 金牛、处女、摩羯 | 稳定、实际、耐心 |
| 💨 风 | 双子、天秤、水瓶 | 思维、交流、社交 |
| 💧 水 | 巨蟹、天蝎、双鱼 | 情感、直觉、敏感 |

#### 三种极性

| 极性 | 星座 | 特点 |
|-----|------|------|
| 主动（开创） | 白羊、巨蟹、天秤、摩羯 | 发起、推动、领导 |
| 固定（稳定） | 金牛、狮子、天蝎、水瓶 | 坚持、深化、巩固 |
| 变动（适应） | 双子、处女、射手、双鱼 | 灵活、调整、适应 |

#### 元素相生（团队加成）

```
风助火：风象助燃火象
水润土：水象滋润土象

相生关系（团队合作）：
💨风 → 🔥火 → 🌍土 ← 💧水
```

| 关系 | 团队战力加成 | 说明 |
|-----|-------------|------|
| **同元素** | +20% | 深度理解，默契配合 |
| **风助火** | +15% | 风象助燃火象，创意爆发 |
| **水润土** | +15% | 水象滋润土象，稳定成长 |
| **其他组合** | +5% | 普通合作 |

#### 对宫星座（战斗对抗）

| 对宫组合 | 元素 | 关系主题 |
|---------|------|---------|
| 白羊 ↔ 天秤 | 火 ↔ 风 | 自我 vs 他人 |
| 金牛 ↔ 天蝎 | 土 ↔ 水 | 物质 vs 情感 |
| 双子 ↔ 射手 | 风 ↔ 火 | 细节 vs 远景 |
| 巨蟹 ↔ 摩羯 | 水 ↔ 土 | 家庭 vs 事业 |
| 狮子 ↔ 水瓶 | 火 ↔ 风 | 个人 vs 群体 |
| 处女 ↔ 双鱼 | 土 ↔ 水 | 现实 vs 理想 |

#### 战斗加成规则

| 关系类型 | 攻击伤害加成 | 说明 |
|---------|-------------|------|
| **对宫星座** | +30% | 激烈对抗，互相伤害 |
| **同极性竞争** | +10% | 争夺主导权 |
| **元素相克** | +5% | 火水相冲、土风相冲 |

**注意：相生关系仅在团队合作时生效，战斗中无加成**

### 3.8 五行系统

#### 核心概念
玩家通过输入生辰八字确定五行主属性，装备和技能也附带五行属性，战斗时触发五行相生相克效果。

#### 五行获取
```
玩家输入生日（年月日时）
    ↓
系统计算八字（天干地支）
    ↓
统计五行出现次数
    ↓
确定主属性（出现最多的五行）
```

#### 五行对应天干地支

| 天干 | 五行 | 地支 | 五行 |
|-----|------|-----|------|
| 甲、乙 | 木 | 寅、卯 | 木 |
| 丙、丁 | 火 | 巳、午 | 火 |
| 戊、己 | 土 | 辰、戌、丑、未 | 土 |
| 庚、辛 | 金 | 申、酉 | 金 |
| 壬、癸 | 水 | 亥、子 | 水 |

#### 五行相生相克

```
相生（生助关系）：
木 → 火 → 土 → 金 → 水 → 木

相克（克制关系）：
木 → 土 → 水 → 火 → 金 → 木
```

#### 战斗效果

| 关系 | 攻击方 | 防御方 |
|-----|-------|-------|
| **相克** | 伤害 +30% | 防御 -20% |
| **相生** | 伤害 -10% | 防御 +20% |
| **同行** | 正常 | 正常 |
| **被克** | 伤害 -20% | 防御 +10% |

#### 装备五行属性

| 装备类型 | 可附带五行 | 特殊效果 |
|---------|----------|---------|
| 武器 | 金、火、木 | 攻击加成、暴击、穿透 |
| 防具 | 土、水 | 防御加成、减伤、反伤 |
| 饰品 | 全五行 | 特殊效果、属性加成 |

#### 技能五行属性

| 五行 | 技能类型 | 特效 |
|-----|---------|------|
| 木 | 治疗、增益 | 持续恢复、成长 |
| 火 | 攻击、暴击 | 燃烧、爆发 |
| 土 | 防御、控制 | 护盾、定身 |
| 金 | 穿透、暴伤 | 破甲、斩杀 |
| 水 | 治疗、净化 | 回复、解控 |

#### 五行战斗计算

```
实际伤害 = 基础伤害 × 五行系数 × 技能系数

五行系数：
- 相克：1.3（克制方）/ 0.8（被克方）
- 相生：0.9（生助方）/ 1.2（被生方）
- 同行：1.0
```

### 3.8 星座匹配系统

#### 匹配规则
根据星座四元素进行匹配，匹配度影响战力加成和合作收益。

| 匹配等级 | 条件 | 战力加成 | 贸易加成 |
|---------|------|---------|---------|
| 最佳匹配 | 跨元素最佳配对 | +20% | +20% |
| 良好匹配 | 同元素星座 | +10% | +10% |
| 普通匹配 | 其他组合 | +5% | +5% |

#### 四元素匹配表
```
🔥 火象（白羊、狮子、射手）
   最佳匹配：风象（双子、天秤、水瓶）
   良好匹配：火象

🌍 土象（金牛、处女、摩羯）
   最佳匹配：水象（巨蟹、天蝎、双鱼）
   良好匹配：土象

💨 风象（双子、天秤、水瓶）
   最佳匹配：火象（白羊、狮子、射手）
   良好匹配：风象

💧 水象（巨蟹、天蝎、双鱼）
   最佳匹配：土象（金牛、处女、摩羯）
   良好匹配：水象
```

#### 战力加成应用场景
- 好友组队战斗：双方获得匹配战力加成
- 战队成员：战队整体战力根据成员匹配度计算加成
- 联盟战争：联盟内匹配度高的成员协同作战有额外加成

### 3.8 战队系统

#### 核心概念
玩家可以创建或加入战队，战队成员根据星座匹配度获得战力加成。战队战斗是团队PVP玩法。

#### 战队创建
- 玩家等级达到 Lv.5 解锁
- 消耗 1000 金币创建
- 战队人数上限：5人（初始），可升级到10人

#### 战队匹配加成
```
战队战力 = Σ(成员战力) × 匹配加成系数

匹配加成系数计算：
- 统计所有成员两两匹配度
- 最佳匹配对数占比 × 20%
- 良好匹配对数占比 × 10%
- 普通匹配对数占比 × 5%

示例：
战队5人，共10对关系
最佳匹配：4对（40%）→ +8%
良好匹配：3对（30%）→ +3%
普通匹配：3对（30%）→ +1.5%
总加成：+12.5%
```

#### 战队功能

| 功能 | 说明 |
|-----|------|
| 战队创建 | 创建战队，设置名称和徽章 |
| 成员招募 | 搜索招募匹配度高的成员 |
| 战队匹配 | 系统自动匹配战力相近的战队 |
| 战队战斗 | 5v5团队战斗，抢夺资源和荣誉 |
| 战队排名 | 根据战绩排名，发放奖励 |
| 战队商店 | 使用战队积分兑换道具 |

#### 战队战斗规则
- 每周可发起3次战队战
- 战斗采用回合制，双方轮流出战
- 胜利方获得：战队积分、资源奖励、荣誉值
- 失败方：无惩罚，但失去一次挑战机会

#### 战队推荐系统
系统根据星座匹配度推荐潜在队友：
```
推荐优先级：
1. 星座最佳匹配
2. 战力相近
3. 活跃度高
4. 好友优先
```

## 四、核心创新点

### 4.1 六位一体融合
```
情绪资源 + 情绪卡片 + 时间货币 + 星座星域 + 社交航道 + 领地战争 = 星座帝国
```

### 4.2 独特玩法组合
- **情绪经济**：情绪成为可交易的动态资源
- **情绪卡片**：真实社交内容产生情绪价值，路人投票定价
- **时间金融**：时间可存取借贷，形成金融体系
- **星座星域**：出生区域影响属性，星座匹配影响社交收益
- **社交基建**：好友关系即航道，互动即流量
- **宇宙贸易**：每个玩家宇宙独特，差异化产生贸易
- **星际战争**：抢星系、抢装备、抢情绪资源

### 4.3 深度社交绑定
- 好友越多，航道越多，贸易机会越多
- 好友互动决定航道流量
- 好友可以驻军、担保、共建
- 星座匹配度影响合作收益
- 情绪卡片分享拉票形成病毒传播
- 联盟形成星际帝国

```go
// 情绪价格受供需影响
func calculateEmotionPrice(emotionType string) float64 {
    totalSupply := getTotalSupply(emotionType)
    totalDemand := getTotalDemand(emotionType)
    basePrice := getBasePrice(emotionType)
    
    // 供需比例影响价格
    ratio := float64(totalDemand) / float64(totalSupply)
    return basePrice * ratio * volatilityFactor
}
```

#### 时间利息计算
```go
// 时间银行利息
func calculateTimeInterest(deposit int64, days int) int64 {
    rate := 0.05 // 日利率 5%
    return int64(float64(deposit) * math.Pow(1+rate, float64(days)))
}
```

#### 战力计算
```go
// 综合战力 = 基础战力 + 装备加成 + 情绪加成
func calculatePower(user *User) int64 {
    basePower := user.Level * 100
    equipmentBonus := calculateEquipmentBonus(user.Equipments)
    emotionBonus := calculateEmotionBonus(user.Emotions)
    return basePower + equipmentBonus + emotionBonus
}
```
