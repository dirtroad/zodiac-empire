# 时空晶体系统设计

## 获取方式

| 方式 | 数量 | 说明 |
|-----|------|------|
| 在线挂机 | 1/小时 | 自动累积，速度慢 |
| 点赞奖励 | 1/赞 | 别人给你的动态点赞 |
| 每日登录 | +10 | 每日首次登录奖励 |
| 任务奖励 | 5-50 | 完成日常任务 |
| 充值购买 | 10/元 | 人民币购买 |

## 存储系统

### 时空银行
- 存入时空晶体
- 每日利息：0.1%（存1000每天得1个）
- 随时存取

### 在线计时
```
在线时间追踪：
- 每分钟检测一次
- 累计满60分钟 → +1时空晶体
- 显示当前在线时长
```
