# 星座帝国 - 地图系统设计

> 双地图系统详细设计 v1.0

---

## 一、地图系统概述

### 1.1 设计理念
采用**双地图系统**，结合真实地理位置（LBS）与虚拟星座星域，创造独特的游戏体验。

### 1.2 双地图架构

| 地图类型 | 基础 | 消耗 | 用途 |
|---------|------|------|------|
| **本地微信地图** | 真实地理位置（LBS） | 免费 | 本地社交、附近玩家、本地资源 |
| **星座地图** | 虚拟星座星域 | **需要时空晶体** | 高级玩法、跨地域交互、稀有资源 |

### 2.4 地盘与建筑物占领系统

#### 占领规则

```
本地地图上的地盘和建筑物：
- 灰色图标 = 未被占领（可占领）
- 绿色图标 = 自己占领（产生收益）
- 蓝色图标 = 好友占领（可贸易）
- 红色图标 = 他人占领（可抢夺）

占领条件：
- 距离：3km以内
- 状态：未被占领
- 消耗：行动力×1
```

#### 地盘类型

| 地盘类型 | 产出 | 占领消耗 | 收益周期 |
|---------|------|---------|---------|
| 公园 | 快乐+10/h | 行动力×1 | 永久 |
| 商场 | 金币+50/h | 行动力×2 | 永久 |
| 学校 | 智慧+15/h | 行动力×2 | 永久 |
| 体育馆 | 激情+20/h | 行动力×3 | 永久 |
| 医院 | 平静+15/h | 行动力×2 | 永久 |

#### 建筑物类型

| 建筑物类型 | 产出 | 占领消耗 | 特殊效果 |
|-----------|------|---------|---------|
| 咖啡店 | 金币+100/h | 行动力×3 | 快乐产出+10% |
| 健身房 | 激情+30/h | 行动力×3 | 战力+2% |
| 书店 | 智慧+25/h | 行动力×3 | 合成成功率+3% |
| 银行 | 金币+200/h | 行动力×5 | 时间利息+5% |
| 科技园 | 智慧+50/h | 行动力×5 | 研究速度+10% |

#### 占领收益

```
收益计算：
- 基础产出：地盘/建筑物固定产出
- 星座加成：根据星座获得额外加成
- 时间累积：需要主动收取

收取方式：
- 手动收取：点击地盘收取
- 一键收取：收取所有已占领地盘收益
- 自动收取：VIP特权，自动到账
```

#### 占领数量限制

| 等级 | 可占领地盘数 | 可占领建筑物数 |
|-----|-------------|--------------|
| 1-5 | 3 | 1 |
| 6-10 | 5 | 2 |
| 11-20 | 8 | 3 |
| 21-30 | 12 | 5 |
| 31-50 | 18 | 8 |
| 50+ | 25 | 10 |

#### 被抢夺规则

```
抢夺条件：
- 目标必须被其他玩家占领
- 抢夺消耗：行动力×3
- 抢夺成功：获得该地盘/建筑物的所有权

⚠️ 无保护时间：占领后随时可能被抢夺！

抢夺机制：
- 战力对比：抢夺者战力 vs 占领者战力
- 战力高者获胜
- 可申请好友驻军帮助防守

抢夺收益：
- 成功抢夺：获得该地盘已累积的50%资源
- 被抢夺：损失已累积的50%资源 + 产生恐惧情绪
```

#### 防守策略

```
保卫地盘的方式：

1. 提升战力
   - 升级装备
   - 收集情绪资源
   - 提升等级

2. 好友驻军
   - 邀请好友派兵驻守
   - 驻军战力叠加到防守战力
   - 驻军好友获得分成

3. 防御建筑
   - 在地盘上建造防御塔
   - 增加防守战力加成

4. 联盟保护
   - 加入联盟
   - 联盟成员可协防
   - 联盟领地有额外加成

5. 🆕 收留流浪者
   - 收留没有地盘的玩家
   - 流浪者帮助保卫地盘
   - 双方形成"领主-家臣"关系

战力计算：
防守战力 = 基础战力 + 装备加成 + 情绪加成 + 驻军战力 + 防御建筑加成 + 流浪者战力
```

#### 流浪者系统

```
流浪者定义：
- 没有占领任何地盘的玩家
- 地盘全部被抢夺的玩家
- 新注册还未占领地盘的玩家

流浪者状态：
- 无法获得地盘产出
- 只能采集公共资源
- 战力加成-20%
- 可被其他玩家"收留"

收留机制：
┌─────────────────────────────────────────────────────┐
│              收留流浪者                             │
├─────────────────────────────────────────────────────┤
│                                                     │
│  流浪者：星尘 (Lv.12)                               │
│  战力：3,500                                        │
│  星座：狮子座                                       │
│                                                     │
│  收留条件：                                         │
│  - 提供庇护所（1个地盘供其驻守）                    │
│  - 每日提供基础资源（金币×100）                     │
│                                                     │
│  收留收益：                                         │
│  - 流浪者战力加入你的防守战力                       │
│  - 流浪者可帮你收取资源（分成20%）                  │
│  - 流浪者可帮你攻击敌人（获得战利品分成）           │
│                                                     │
│  流浪者收益：                                       │
│  - 获得庇护所和基础资源                             │
│  - 战力惩罚解除                                     │
│  - 可积累资源重新占领地盘                           │
│                                                     │
│  [收留]  [查看其他流浪者]                           │
│                                                     │
└─────────────────────────────────────────────────────┘

收留规则：
- 每个玩家最多收留 3 名流浪者
- 流浪者可随时离开，寻找新领主
- 领主可随时驱逐流浪者
- 流浪者积累足够资源后可"独立"，重新占领地盘

流浪者等级：
| 等级 | 条件 | 分成比例 |
|-----|------|---------|
| 仆从 | 刚收留 | 20% |
| 家臣 | 收留7天 | 30% |
| 亲信 | 收留30天 | 40% |
| 独立 | 积累5000金币 | 可离开独立 |

#### 星座匹配加成

```
收留/结盟时，星座匹配度影响加成：

| 匹配等级 | 战力加成 | 资源分成 | 特殊效果 |
|---------|---------|---------|---------|
| 最佳匹配 | +30% | +10% | 解锁"星座共鸣"技能 |
| 良好匹配 | +20% | +5% | 无 |
| 普通匹配 | +10% | 0% | 无 |
| 不匹配 | 0% | -5% | 流浪者可能叛变 |

星座匹配表：
🔥 火象（白羊、狮子、射手）↔ 💨 风象（双子、天秤、水瓶）= 最佳匹配
🌍 土象（金牛、处女、摩羯）↔ 💧 水象（巨蟹、天蝎、双鱼）= 最佳匹配
同元素 = 良好匹配
其他 = 普通匹配
对宫星座 = 不匹配（如白羊↔天秤）

星座共鸣技能（最佳匹配专属）：
- 火风共鸣：攻击时10%概率触发"烈焰风暴"，伤害+50%
- 土水共鸣：防守时10%概率触发"大地护盾"，减伤30%
- 可通过收留多个最佳匹配星座的流浪者叠加效果
```

流浪者互动：
- 领主可赠送资源帮助流浪者成长
- 流浪者可主动攻击领主的敌人（获得额外奖励）
- 流浪者被其他领主挖角时，领主可选择加薪挽留
```

### 2.5 时空晶体远程抢占

#### 功能说明

```
远程抢占：使用时空晶体，抢占远处城市的地盘和建筑物

消耗：
- 时空晶体：根据距离计算
- 行动力：×3

计算公式：
晶体消耗 = 基础消耗(50) + 距离(km) × 5

示例：
- 抢占10km外的地盘：50 + 10×5 = 100晶体
- 抢占50km外的地盘：50 + 50×5 = 300晶体
- 抢占100km外的地盘：50 + 100×5 = 550晶体
```

## 五、地图切换流程

## 八、地图性能优化

### 8.1 分块加载

```
地图分块：
- 将地图划分为16×16的区块
- 只加载视野内的区块
- 预加载相邻区块
- 卸载远离视野的区块
```

### 8.2 数据缓存

```
缓存策略：
- 星域数据：缓存1小时
- 玩家位置：实时更新
- 资源点：缓存5分钟
- 事件：实时推送
```

### 8.3 渲染优化

```
渲染策略：
- 使用对象池管理图标
- 批量渲染相同类型元素
- 使用LOD（细节层次）技术
- 减少重绘区域
```

## 十、地图与时空晶体关联

### 10.1 穿梭消耗规则

```
穿梭消耗计算：
base_cost = region.base_cost
distance_bonus = calculate_distance(from, to) * 0.1
element_relation = get_element_relation(user_element, target_element)

final_cost = base_cost + distance_bonus + element_relation

元素关系修正：
- 同元素：-10晶体
- 相生元素：-5晶体
- 相克元素：+10晶体
```

### 10.2 穿梭冷却

```
穿梭冷却：
- 普通穿梭：无冷却
- 连续穿梭：每次+5分钟冷却
- 冷却上限：30分钟
- 可用晶体消除冷却：50晶体
```

## 十一、技术实现方案

### 11.1 微信小游戏地图限制

```
微信小游戏限制：
- 无原生地图组件（小程序有map组件，小游戏没有）
- wx.openLocation 只能显示单一标记
- 无法在微信地图上添加自定义标记

解决方案：列表模式 + Canvas地图
```

### 11.2 第一阶段：列表模式

```
实现方式：
1. wx.getLocation 获取用户GPS坐标
2. 后端根据坐标查询附近玩家/地盘
3. 前端列表展示，按距离排序
4. 点击列表项进行操作

优点：
- 开发快速，可立即验证玩法
- 无需处理地图渲染复杂度
- 用户体验清晰

列表展示内容：
- 地盘/建筑物名称
- 距离（多少米/公里）
- 当前领主（头像、昵称、等级）
- 状态（可占领/已被占领/可抢夺）
- 操作按钮（占领/抢夺/结盟/查看）
```

### 11.3 第二阶段：Canvas地图

```
实现方式：
1. 绘制简化地图背景（网格或区域块）
2. 根据GPS坐标在Canvas上绘制标记
3. 支持缩放和拖动
4. 点击标记显示详情弹窗

技术要点：
- 使用Cocos Creator的Graphics组件绘制
- 坐标转换：GPS坐标 → 屏幕坐标
- 视野裁剪：只绘制视野内的标记
```
